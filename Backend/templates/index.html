Model Overview

Our predictive system is powered by a fine-tuned Linear Regression model trained on over 13,320 Bangalore property records. It understands relationships between square footage, location, and configuration to estimate realistic market prices.

The model achieved an R² score of 0.86 on test data — meaning it explains over 86% of price variance with exceptional consistency. Predictions are expressed in lakhs and crores for better real-world readability.

Feature Engineering

Beyond raw data, we introduced intelligent derived metrics such as sqft_per_bhk (efficiency ratio) and price_per_sqft (value normalization). This transformation ensures the model learns structural fairness — identifying whether an apartment is overpriced relative to its size and locality.

1300+ localities were clustered into 240 meaningful regions, preserving location intelligence while preventing data sparsity. These engineered features boost both stability and explainability of predictions.

Outlier Handling

We implemented strong domain-driven outlier filters. Any property that violates natural market logic — such as 1BHK units over 2000 sqft or 8BHK apartments under 600 sqft — is systematically removed.

Additionally, per-location price normalization (mean ± std deviation) ensures consistent comparisons. This refinement significantly reduced skewness, making predictions more dependable and realistic.

Validation & Reliability

Our model’s reliability stems from transparent design. Every prediction undergoes range validation to ensure no property violates area-to-room ratios, preventing false estimates.

Combined with rule-based sanity checks and manual cross-verification on sample data, this framework ensures credibility — making every price output not just accurate but context-aware.

Tech Stack

Built using Python, Flask, and scikit-learn, the system integrates machine learning with a responsive, lightweight web interface. The design prioritizes performance, readability, and scalability.

Deployed seamlessly on Render’s free cloud infrastructure, the application is optimized for accessibility without compromising quality — providing users with an instant, elegant predictive experience.
